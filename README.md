# Instagram Influencer Automation using n8n

## Overview
This workflow automates Instagram influencer data extraction based on hashtag search.

## Features
- Automated hashtag scraping
- Duplicate URL removal
- AI niche categorization
- Fake influencer detection
- Engagement quality analysis
- Niche relevance scoring
- Automated outreach message generation
- Excel report generation
- Scheduled daily execution
- Email delivery automation
- Google Sheets integration

## Workflow Architecture
Schedule Trigger → HTTP Request → Data Processing → XLSX Conversion → Email Delivery

## Tech Stack
- n8n
- Apify API
- JavaScript
- Google Sheets
- SMTP Email

## Output
Excel report containing influencer analytics
